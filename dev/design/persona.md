## Phase 4: ペルソナ分析 → デザイン決定

分析結果をIBM Carbon準拠のデザイン決定に変換してください。

### 4-1. カラーパレット選定

ペルソナ属性ごとの方針を確認し、選定根拠を記述してください：

| ペルソナ属性 | デザイン方針 |
|---|---|
| 年齢層40歳以上 | 文字コントラスト高め（7:1以上推奨）、フォントサイズ大きめ |
| ITリテラシー低 | シンプルな色使い（3色以内）、視覚的ヒエラルキーを明確に |
| B2Bプロフェッショナル | ネイビー・グレー系で信頼感を演出 |
| B2Cコンシューマー | アクセントカラーで親しみやすさ・エネルギーを表現 |
| モバイル中心 | 指操作を考慮したコントラスト・余白の確保 |

## 参照: IBM Carbon設計原則との対応

| IBM Carbonの原則 | ペルソナ分析との接続ポイント |
|---|---|
| Accessibility by Default | 年齢・障害に応じたコントラスト・サイズ設定 |
| Consistency | ペルソナの学習コスト最小化 → 一貫したパターン |
| Clarity | 「わかりにくい」フラストレーションへの対応 |
| Efficiency | ゴール達成の最短経路を設計 |
| Designed with the user | ペルソナ実在人物を常に想定して判断 |
