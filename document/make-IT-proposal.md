# IT系システム開発に特化した詳細ビジネス企画書作成プロンプト

以下のテンプレートは、IT系システム開発ビジネス案に特化した包括的な企画書を作成するためのものです。システム開発プロジェクトの特殊性を考慮し、技術的詳細、開発アプローチ、導入プロセス、および投資対効果の分析に重点を置いています。ソフトウェア開発、システムインテグレーション、AIソリューション導入、プラットフォーム構築など、あらゆるITプロジェクトに適応可能です。

企画書はマークダウン形式で出力し、タイトル、目次、各セクションの説明と詳細を含む完全なドキュメントとして構成してください。システム構成図、データフロー、技術アーキテクチャなどの技術的側面と、ビジネスインパクト、ROI、市場機会といったビジネス側面の両方をバランスよく含みます。

## 入力情報

以下の情報を提供してください：

1. **企画タイトル**: [製品・サービス名]
2. **組織名**: [あなたの組織名]
3. **作成者名とポジション**: [名前 - 組織内での役職]
4. **主要なターゲット市場**: [特定の業界、セクター、顧客層]
5. **解決する主要な課題**: [この企画が解決しようとしている主要な問題や課題]
6. **差別化要素**: [競合に対する主な差別化ポイント、3-5点]
7. **予算範囲（オプション）**: [おおよその開発・実装予算]
8. **実装タイムライン（オプション）**: [主要なマイルストーンと期間]

## 出力形式

AIは以下の構造に従った完全な企画書を生成します。各セクションは詳細な内容、分析、数値データ（可能な場合）を含み、専門家レベルの詳細さと洞察を提供します。AIは関連する事実、市場データ、技術トレンドを調査し、企画書に組み込みます。

## 出力ガイドライン

1. 全体の企画書はマークダウン形式で、きちんと構造化されたアーティファクトとして提供します
2. 各セクションには具体的でデータに基づいた内容を含めます
3. 表やリストを活用して情報を整理して提示します
4. 専門的で論理的な文体を使用し、提案の信頼性を高めます
5. 可能な限り最新のデータや市場トレンドを参照します
6. 現実的かつ実行可能な提案を行います
7. AIが調査できる市場データや技術トレンドはAIが検索・分析します
8. ユーザーは自社特有の情報のみを提供すれば十分です

## 例：必要な入力例

```
1. 企画タイトル: AIを活用した医療画像診断支援システム
2. 組織名: ヘルステック株式会社
3. 作成者名とポジション: 山田太郎 - 新規事業開発部長
4. 主要なターゲット市場: 日本国内の中小規模病院と診療所
5. 解決する主要な課題: 放射線科医不足による画像診断の遅延と精度のばらつき
6. 差別化要素: 
   - 日本語医療用語に最適化された独自AI
   - 院内システムとの容易な連携
   - 段階的な導入が可能なモジュール型設計
   - プライバシー保護のためのオンプレミス処理
7. 予算範囲: 初期開発 5000万円、年間運用 2000万円
8. 実装タイムライン: 基本モジュール 6ヶ月、拡張機能 12ヶ月
```

これらの情報に基づいて、AIが包括的なビジネス企画書を生成します。
